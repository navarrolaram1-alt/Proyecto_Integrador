# Portfolio Personal - Lara Navarro

## Descripción

Este proyecto consiste en el desarrollo de un portfolio personal realizado como trabajo integrador para la materia Programación III. El sitio presenta información sobre mi formación, habilidades e intereses en el desarrollo de software, además de un formulario de contacto y funcionalidades interactivas implementadas con JavaScript.

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- Git
- GitHub

## Funcionalidades implementadas

- Navegación entre secciones del portfolio.
- Sección de proyectos generada dinámicamente con JavaScript.
- Formulario de contacto con validación de campos obligatorios.
- Verificación del formato del correo electrónico.
- Contador de caracteres en el campo de mensaje.
- Manipulación del DOM mediante la creación y modificación de elementos.
- Manejo de errores utilizando `try...catch`.

## Conceptos de JavaScript aplicados

- Funciones.
- Eventos (`click`, `submit` e `input`).
- Arrays.
- Manipulación del DOM.
- Validaciones.
- Manejo de excepciones.
